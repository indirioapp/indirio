# Indirio App 📥

Welcome to the official GitHub organization of **Indirio**—a modern, high-performance, and open-source media downloading ecosystem.

## 🚀 What is Indirio?

Indirio is a fast, ad-free, and privacy-focused media downloader designed to retrieve high-quality video, audio, and subtitle streams from major social platforms including **YouTube, TikTok, Instagram, and Twitter (X)**. 

Unlike traditional downloaders cluttered with intrusive pop-up ads and malicious redirects, Indirio provides a clean, self-hostable, and lightning-fast alternative built on top of modern technology.

## 🛠️ Main Projects

*   **[indirio](https://github.com/indirioapp/indirio)**: The core Next.js web application, featuring a responsive futuristic user interface, multi-format media extraction, playlist compilation, and automated disk space optimization.

## 🌟 Our Core Values

*   **Ad-Free Experience**: No pop-ups, no malicious scripts, just pure downloads.
*   **Privacy-First**: No tracking or data logging. Downloads are processed and automatically wiped from server disk after 2 minutes.
*   **High Performance**: Powered by Next.js Serverless streams, yt-dlp, and FFmpeg for optimal speed.
*   **Open Source**: Freely auditable, self-hostable, and community-driven.
